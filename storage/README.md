# Storage Directory

This directory contains persistent data for the Discord bot.

All JSON files in this directory are automatically created and managed by the bot:
- `afkData.json` - AFK status tracking
- `blacklist.json` - Blacklisted users
- `chatMemory.json` - Chat conversation memory
- `commandUsage.json` - Command usage statistics
- `convoSummaries.json` - Conversation summaries
- `memory.json` - Bot memory data
- `msgData.json` - Message tracking data
- `userProfiles.json` - User profile information

**Note:** These files are excluded from git commits to protect user privacy.
